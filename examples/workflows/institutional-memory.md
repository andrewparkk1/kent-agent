# Institutional Memory

Weekly scan for "why did we decide…" or "does anyone remember…" questions floating around Slack and email, cross-referenced against your own decision log so you can drop in the answer before someone else has to dig for it.

Create a workflow that runs every Friday at 4pm. Scan Slack and email from the past week for questions like "why did we decide X", "does anyone remember when we...", "who was on that call where...", "what was the reason we went with...", "do we have docs on...", "has anyone ever looked at...". Also catch implicit versions — someone asking a question that the team already answered months ago in a different thread. For each one, look up the answer by: (1) searching my memories for relevant decisions, (2) cross-referencing the decision log from past meetings and threads, (3) checking Granola transcripts for when the topic came up, (4) looking at linked docs or PRs that explain the rationale. If you find a confident answer, draft a short reply that includes the decision, the context of when/why it was made, who was involved, and a link or reference to the source. Match the tone of the original thread — Slack reply vs. email — and save as a draft, don't auto-send. If you can't find a confident answer, flag it as "institutional gap" with a note on what would be needed to answer it (e.g. "this predates any of my synced data" or "no decision log entry exists for this"). Output a weekly summary: questions answered, drafts ready, and institutional gaps worth filling in. Save any new decisions or rationales I surface to memory so the next scan gets sharper.
