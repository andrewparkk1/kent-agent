# Daily Lesson

Learn one concept a day on any topic you choose, building from basics to advanced over time.

Create a workflow that runs every weekday at 9am and teaches me one concept about [your topic here]. Start from the absolute basics and work up. Each lesson should build on the last one. Explain things in plain English with a real-world analogy, then give me the technical version with a concrete example. Tell me how it connects to what I learned before, and give me something small to try or think about. Keep it to a 5-minute read. Save each lesson to memory so you know where I left off.
