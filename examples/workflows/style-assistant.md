# Style Assistant

Learns your taste in clothes over time and searches the web for things that actually match your style.

Create a workflow that runs every Saturday at 10am and suggests clothes for me. Look at my order confirmation emails, any messages about shopping or outfits, and what I've been browsing to learn my style over time. Search the web for 3-5 specific items each week — not vague stuff like "a nice jacket" but specific things like "navy linen camp collar shirt." Include links to where I can actually buy them and how much they cost. If you notice gaps in my wardrobe, point them out. Save everything you learn about my preferences to memory so your suggestions get better over time. If you don't know my style yet, just ask me a few questions.
